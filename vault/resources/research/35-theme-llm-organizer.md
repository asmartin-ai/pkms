# 35 — Theme: LLM as organizer

**Why this matters for the build:** your Notion postmortem ("without an LLM to help organize, I got lost in perfectionism") turns out to be the central finding of the whole research program. Mark each: ✅ / ❌ / ❓.

1. **LLM-as-organizer is the first structurally viable ADHD PKM architecture.** Pre-LLM GitHub has ~zero ADHD PKM tools — organization-on-write always died; the 2026 "markdown vault + agent" wave is the first fit ([[14-github]] F17, [[11-hn]] F18).
   **React:** ✅ / ❌ / ❓ —
2. **The division of labor: human dumps, LLM structures, index makes it queryable.** Voice/text/clips land raw; the agent extracts knowledge and tasks ([[11-hn]] F18, [[13-youtube]] F8, F9).
   **React:** ✅ / ❌ / ❓ —
3. **AI lives as a regenerable layer over plain files — never as the app holding the notes.** Reor (AI-app) is archived; Smart Connections / Copilot / basic-memory (AI-layers) all live. File over app ([[18-landscape]] LS2–LS6, LS12).
   **React:** ✅ / ❌ / ❓ —
4. **Interaction grammar: present-then-ask.** The agent proposes (a classification, a plan, 3 options) and asks at most one clarifying question; you pick. Never a blank prompt, never ask-then-wait — you're "extremely proficient at identifying the best from a list" ([[12-reddit]] F23, F9; [[19-seed-links]] SD6).
   **React:** ✅ / ❌ / ❓ —
5. **The LLM closes loops, not just summarizes:** "X was in progress — finished?" plus exactly one concrete next step. That's the accountability layer that broke analysis paralysis in the wild ([[11-hn]] F19).
   **React:** ✅ / ❌ / ❓ —
6. **Passive observer agents die in minutes; agents that hand you the next concrete step get used** ([[19-seed-links]] SD12).
   **React:** ✅ / ❌ / ❓ —
7. **Automation must not manufacture a bigger look-at-later pile.** Auto-organized output gets bounded and surfaced (fixed-size reading sample, overflow auto-archives) ([[11-hn]] F11).
   **React:** ✅ / ❌ / ❓ —
8. **Your briefing-package idiom (context + guardrails + task spec + output location) is the native shape for triage agents** — you already run it with Codex ([[21-job-search-distill]] JS11).
   **React:** ✅ / ❌ / ❓ —
